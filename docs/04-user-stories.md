# User Stories

## Epic: Credit Card Optimization

### Story 1

As a user,

I want to add my credit cards,

So that the application can recommend the best card.

---

### Story 2

As a user,

I want to enter a transaction amount and category,

So that I can receive the best card recommendation.

---

### Story 3

As a user,

I want to understand why a card is recommended,

So that I can trust the recommendation.

---

### Story 4

As a user,

I want to compare reward values across all my cards,

So that I can make informed decisions.

---

### Story 5

As a user,

I want to track my card milestones,

So that I can maximize annual benefits.
