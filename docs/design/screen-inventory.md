# Screen Inventory

## Screen 1 - Landing Page

Purpose:
Explain product value and allow users to start.

Components:

* Hero section
* Features
* CTA button

---

## Screen 2 - Dashboard

Purpose:
View saved cards and recommendations.

Components:

* Card summary
* Quick stats
* Recent recommendations

---

## Screen 3 - Add Card

Purpose:
Add a credit card.

Fields:

* Card name
* Issuer
* Reward rates
* Annual fee

---

## Screen 4 - Optimize Transaction

Purpose:
Get card recommendation.

Inputs:

* Amount
* Category
* Merchant

Output:

* Best card
* Reward value
* Reason

---

## Screen 5 - Recommendation Result

Purpose:
Compare all cards.

Components:

* Ranking
* Reward comparison
* Recommendation explanation
